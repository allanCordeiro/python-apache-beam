# Casos de Dengue

Repo com o estudo de caso proposto no Curso da Alura: https://cursos.alura.com.br/course/apache-beam-data-pipeline-python

Foram feitos pouquíssimas refatorações no código proposto na aula. A idéia era só entender o funcionamento dos pipelines no Apache Beam.

Caso tenha interesse e queira testar o código:

- ative um ambiente virtual;
- instale o requirements: pip install -r requirements.txt
- Descompacte o arquivo *chuvas.tar.xz* (não é necessário caso queira rodar apenas os samples)
- execute o arquivo main.py.
